## ComicRead
一款漫画阅读APP，独立开发，数据来源于腾讯动漫网，通过爬虫抓包获取

## 功能实现
使用Jsoup框架解析爬取腾讯动漫网漫画数据，实现漫画分类，热门漫画，漫画查询，看漫画等功能。
